Installation & Setup Guide

The application requires the use of Docker to run. The first part of the instructions will cover the installation of the packages for docker & docker-compose.

The second part of the instructions will cover running the application and basic management of the containers.

Official Docker documentation can be found at the links below.

  • Docker Engine https://docs.docker.com/install/
  • Docker Compose https://docs.docker.com/compose/install/

Prerequisites

There are a couple of things that you will need to make this install as easy as possible.

  1. A copy of the latest docker-compose.yml file from the dev team.
  2. .env file containing your game server information.
  3. broadcasts.txt file for sending premade broadcast messaages to your server automatically
  4. db_data, logs & pw folders
  5. .htpasswd file to protect your rcon from unauthorised access.

Using the link below download the .zip file containing items 1-4 of the above list and extract the contents to a directory of your choosing i.e. /home/user.name/
